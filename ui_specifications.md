# UI Specifications

LVGL-based UI for ESK8 Dashboard.

## Layout
*   **Center**: Primary readout (Motor ERPM)
*   **Top**: Battery voltage (V) and percentage (%)
*   **Bottom Left**: Total current draw (A)
*   **Bottom Right**: ESC Temperature (°C)

## Styling
*   **Colors**: Black background, white text
*   **Warnings**: Red text for alerts (Temp > 75°C or Battery < 32.0V)
*   **Units**: Metric exclusively
